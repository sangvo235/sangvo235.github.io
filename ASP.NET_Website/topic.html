What is ASP.NET?




ASP.NET is an open-source web application framework based on C#. It is used to build web applications on the .NET framework.

  • 1998 – Classic ASP (Active Server Pages) was released by Microsoft as their first server-side scripting language
  • 2002 – Version 1.0 of ASP.Net was released by Microsoft in 2002 and was the successor of Classic ASP

3 Reasons why ASP.NET is great!




  1. It is a lot simpler to maintain & update, which results in an affordable and highly-competent framework
  2. ASP.NET is created and supported by Microsoft, being one of the leading companies in the globe there is higher level of safety and security assurance for users
  3. It offers customization in the data storage where you can easily distinguish identity information storage and security information
